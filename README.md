# Свадебное приглашение — Елизавета & Денис

https://inv.d-corp.ru/

## Блоки сайта

**Шапка (Hero)** — parallax-фон, имя гостя из токена (или скелетон при загрузке), дата 12.09.2026, кнопка подтверждения.

**Навигация** — фиксированный navbar, прозрачный → backdrop-blur при скролле, пункты: История, Детали, Дресс-код, Программа, Вопросы, Подтвердить.

**Наша история** — три абзаца с staggered fade-in, фото с розовой декоративной подложкой.

**Где и Когда** — три glassmorphism-карточки: дата (сбор в 14:50), место (ресторан «Охотник»), парковка.

**Как добраться** — две Яндекс.Карты рядом: точка сбора для трансфера и ресторан.

**Дресс-код** — 4 цветных круга с текстурой noise и ободком, палитра: Пудровый, Шампань, Бежевый, Тёмно-зелёный.

**Программа дня** — таймлайн 14:50–23:00 (сбор, фуршет, церемония, ужин, afterparty, обратный трансфер), alternating left/right.

**Таймер** — обратный отсчёт до 12.09.2026 в glassmorphism-блоках.

**Частые вопросы** — аккордеон с 5 вопросами.

**Подтверждение** — форма RSVP: присутствие (да/нет), гости, напиток (из Google Sheets), трансфер, пожелания. Guest auth через токен в URL, кэширование в localStorage.

**Футер** — контакты организаторов: Татьяна 8 905 850 80 50, Ксения 8 919 565 49 29.

## Стек
